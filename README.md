# agentic-dev-kit

A centralized catalog of Claude Code skills (and, later, slash commands) that teams copy
into their own projects to get consistent, agentic frontend development.

The current contents are **one opinionated pipeline** for a specific stack — Turborepo +
pnpm, NextJS web, bare React Native CLI, Figma MCP, TanStack Query, a Frappe backend.
The skills are coupled and meant to be adopted together, not cherry-picked.
